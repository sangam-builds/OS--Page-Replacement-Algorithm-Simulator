# Page Replacement Algorithm Simulator

A comprehensive web-based simulator for demonstrating and comparing different page replacement algorithms used in operating systems memory management.

## Features

- Multiple algorithm implementations: FIFO, LRU, LFU, and Optimal
- Interactive visualization of memory pages
- Real-time statistics and performance metrics
- C-based backend for efficient algorithm execution
- Responsive web interface

## Project Structure

- **Backend**: Express.js server with C algorithms
- **Frontend**: HTML/CSS/JavaScript interactive UI
- **Algorithms**: C implementations of page replacement algorithms
- **Tests**: Unit and integration tests
